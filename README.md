# Bouncy-Ball
A simple Java Game inspired by OG game **Breakout**.

## Getting Started
### Requirements
* Java 8 or later
* Gradle (optional, for build automation)

### Clone the Repository
``` bash
git clone https://github.com/Brijendra-Singh2003/Bouncy-Ball.git
cd Bouncy-Ball
```

### Run with Gradle
``` bash
gradle clean build
gradle run
```
